## Legion Stage Session Workflow

This skill covers a 4-stage production workflow for electronic/synth-focused music in the Legion Stage DAW.

### Stages

1. **Seed** — Start by laying a backing beat OR recording an inspiring melody. Get something playing to perform against.
2. **Layer** — Loop what you have and add melodic layers (bass, chords, lead, texture) on new tracks.
3. **Shape** — Move clips around to build up and break down the orchestration within a section.
4. **Extend** — Apply the seed→layer→shape process to each section of the full arrangement.

### Key Principles

- Each stage produces output that feeds the next stage
- The workflow is cyclical — Extend loops back through Seed/Layer/Shape for each new section
- Electronic tracks typically follow: Intro → Build → Drop → Breakdown → Second Drop → Outro
- Less is more — two strong layers beat five cluttered ones
- Capture the feeling first, refine later
