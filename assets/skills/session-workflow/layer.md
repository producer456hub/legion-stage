## Goal

Build out your seed into a full-sounding section. Add melodic parts, bass, pads, or textures on new tracks while looping what you have.

## Approach

- Keep the loop running and jam along on a new track
- Record in passes — one new element per track
- Listen for what's missing: low end? movement? texture? rhythm variation?
- Each layer should add something the others don't — don't double up on the same frequency range
- Mute/solo tracks to check how layers interact

## Layering Order (suggestion)

1. **Bass** — anchor the low end, lock to the kick if you have one
2. **Chords/Pad** — fill the mid-range, set the harmonic context
3. **Lead/Melody** — the hook, the thing people remember
4. **Texture** — noise, atmosphere, risers, subtle movement

You don't need all four. Two strong layers can be enough.

## When to Move On

You have 3-6 tracks that sound good together in a loop. The section feels "full" — adding more would clutter it. Move to **Shape**.

## Edge Cases

- If layers clash, try transposing one by an octave or changing the patch
- If it sounds muddy, solo each track — one of them is probably fighting the bass
- Less is more — if in doubt, mute a layer and see if you miss it
